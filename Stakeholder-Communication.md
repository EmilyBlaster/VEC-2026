## Design Direction

We drew inspiration from enterprise-level event sites (think Dreamforce, AWS re:Invent) and adapted that energy to the Intuit for Education brand. The goal was to create something that feels premium, dynamic, and unmistakably "us" -- using our Wild color system as the foundation.

### Key Design Highlights

**Animated Gradient Hero**
The hero section uses a system of six independently animated gradient blobs in our brand Fuchsia, Agave, and Tomato colors. These create a living, breathing backdrop that shifts and morphs continuously -- far more engaging than a static gradient or hero image.

**Curved Text Marquee**
Inspired by Dreamforce's signature curved text section, we have a wide arcing text band that scrolls messaging like "Invest In Your Classroom" and "Free to All Educators" -- a visually striking moment that reinforces our key value propositions as users scroll.

**Community Photo Grid with Stats**
An asymmetric photo mosaic featuring educator imagery with animated stat counters (5,000+ Educators Expected, 20+ Breakout Sessions, 4+ Audience Tracks). This builds social proof and gives a sense of scale.

**Floating 3D Financial Icons**
Our 3D brand icons (piggy bank, diamond, stacked coins) float subtly throughout the page as decorative accents, reinforcing the financial literacy theme without being heavy-handed.

**Real Content from the Event Brief**
The four audience tracks (General Teachers, CTE Teachers, Higher Education, Administrators) are populated with real session topics and room assignments from the Programs Event Brief. This isn't placeholder content -- it reflects our actual event structure.

**Persistent Registration Widget**
A floating countdown timer with "Did we mention FREE?" messaging and a Register Now button follows the user down the right side of the page -- a constant, non-intrusive nudge toward registration.

## Technical Approach

- **Pure HTML/CSS/JS** -- no external frameworks (Bootstrap, Tailwind, etc.)
- **Swoogo-compatible** -- built specifically for drop-in deployment to our event platform
- **Accessibility-first** -- all animations respect `prefers-reduced-motion`, semantic HTML throughout
- **Brand-compliant** -- Avenir Next font loaded locally, all colors via CSS custom properties from the brand palette
- **Responsive** -- adapts across desktop, tablet, and mobile
